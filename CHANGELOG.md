# Changelog

All notable changes to this project are documented in this file.

## [1.0.0] - 2026-04-16

### Added
- CLI `--version` flag to print installed package version and exit.
- README quick start section and release-facing badges.

### Changed
- Package version bumped to `1.0.0`.
- HTTP fetcher User-Agent now resolves version from package metadata to avoid hardcoded drift.
- README and project messaging aligned for v1.0 release.
